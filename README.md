# Puyatas Maestras — Web Assets

Este repositorio contiene scripts JavaScript utilizados en la web de **Puyatas Maestras** para añadir funcionalidades e interacciones personalizadas.

Actualmente se usa para gestionar comportamientos de scroll horizontal en elementos visuales largos.

---

## Contenido

- `horizontal-scroll.js`  
  Convierte el scroll vertical del ratón en desplazamiento horizontal dentro de un contenedor.

---

## Uso

El script actúa sobre cualquier elemento con la clase:

```html
.scroll-horizontal
